API 接口文档

1. 文本改写接口（兼容 5118 格式）

POST /api/rewrite

请求参数（支持两种格式）：

格式1 - 标准格式：

{
  "text": "要改写的文本内容",
  "mode": "standard",
  "model": "claude-3-sonnet-20240229",
  "sim": 1  // 可选，是否返回相似度
}

格式2 - 5118 兼容格式：

{
  "txt": "要改写的文本内容",
  "retype": 1,
  "sim": 1
}

返回格式（5118 兼容）：

{
  "errcode": "0",
  "errmsg": "",
  "data": "改写后的文本内容",
  "like": "0.5521",  // 相似度（如果 sim=1）
  "_meta": {
    "duration": 2.5,
    "usage": {
      "input_tokens": 100,
      "output_tokens": 150
    }
  }
}

cURL 示例：

curl -X POST http://localhost:8000/api/rewrite \
  -H "Content-Type: application/json" \
  -d '{"txt":"要改写的文本","sim":1}'

2. 小旋风集成配置

在小旋风伪原创 API 配置中：

API 地址: http://your-server:8000/api/rewrite
请求方式: POST
请求参数: txt=[content]&sim=1
返回格式: JSON
提取路径: data

小旋风配置示例：

  • API 接口: http://your-server:8000/api/rewrite
  • 请求方式: POST
  • 参数名: txt
  • 参数值: [content] (小旋风变量)
  • 返回提取: JSON 路径 data

3. 错误码说明

  • 0 - 成功
  • 200201 - 传进参数为空
  • 200500 - 内容长度超过5000字符
  • 100203 - API Key 无效
  • 100102 - 请求频率超限

3. 其他接口

  • GET /health - 健康检查
  • GET /api/models - 获取支持的模型列表
  • GET /api/docs - API 文档（JSON）
