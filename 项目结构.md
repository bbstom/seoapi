# 项目结构说明

## 📁 目录结构

```
seoapi-nodejs/
├── config/                      # 配置文件目录
│   └── rewrite-modes.js        # 改写模式配置
├── data/                        # 数据文件目录
│   ├── users.json              # 用户数据

│   ├── index.html              # 主界面
│   └── login.html              # 登录页面
├── node_modules/                # Node.js 依赖包
├── server.js                    # 主服务器文件
├── package.json                 # 项目配置和依赖
├── package-lock.json            # 依赖锁定文件
├── .env                         # 环境变量配置（需自行创建）
├── .env.example                 # 环境变量示例
├── .gitignore                   # Git 忽略文件配置
├── start.bat                    # Windows 启动脚本
├── start.sh                     # Linux 启动脚本
├── stop.sh                      # Linux 停止脚本
├── test-defaults.bat            # 默认配置测试脚本
└── 文档/
    ├── README.md                # 项目说明
    ├── 快速开始.md              # 快速开始指南
    ├── 快速参考.md              # 快速参考指南
    ├── 更新日志.md              # 版本更新记录
    ├── 项目结构.md              # 项目结构说明
    ├── 提示词优化说明.md        # 提示词优化和问题修复
    ├── 默认配置说明.md          # 默认配置功能说明
    ├── 认证系统说明.md          # 用户认证系统说明
    ├── 安全配置说明.md          # 安全和速率限制说明
    ├── 小旋风配置教程.md        # 小旋风集成教程
    ├── Web配置说明.md           # Web 界面使用说明
    ├── 如何自定义改写模式.md    # 自定义模式教程
    ├── AI检测说明.md            # AI 检测风险说明
    ├── API中转说明.md           # API 中转配置说明
    ├── 5118格式说明.md          # 5118 API 格式说明
    └── Linux部署教程.md         # Linux 部署指南
```

## 📄 核心文件说明

### 服务器文件

- **server.js** - 主服务器文件，包含所有 API 路由和业务逻辑
- **package.json** - 项目配置，定义依赖包和脚本命令

### 配置文件

- **config/rewrite-modes.js** - 改写模式配置，可通过 Web 界面修改
- **.env** - 环境变量配置（可选，也可在 Web 界面配置）

### 数据文件

- **data/users.json** - 用户数据（用户名、密码、API Key 等）
- **data/sessions.json** - 登录会话数据

### 前端文件

- **public/index.html** - 主界面（文本改写、模式管理、配置）
- **public/login.html** - 登录页面

### 启动脚本

- **start.bat** - Windows 启动脚本
- **start.sh** - Linux 启动脚本
- **stop.sh** - Linux 停止脚本

### 测试脚本

- **test-defaults.bat** - 测试默认配置功能

## 🔧 常用操作

### 启动服务

```bash
# Windows
start.bat

# Linux
./start.sh
```

### 停止服务

```bash
# Windows
按 Ctrl+C

# Linux
./stop.sh
```

### 测试功能

```bash
# 测试默认配置
test-defaults.bat
```

### 查看日志

服务运行时会在控制台输出日志，包括：
- 请求信息（用户、模式、模型）
- 改写结果（耗时、输出长度）
- 错误信息
