# 快速参考指南

## 🎯 默认配置功能（新增）

### 功能说明

现在可以在 Web 后台为每个用户设置**默认的模型和改写模式**。

当小旋风调用 API 时：
- **指定了参数** → 使用指定的值
- **没有指定参数** → 使用用户配置的默认值

### 配置步骤

1. 访问 `http://localhost:8000`
2. 使用 admin/admin123 登录
3. 点击"配置"标签页
4. 在"默认调用配置"区域设置：
   - 默认模型（推荐：Claude Sonnet 4.5）
   - 默认改写模式（推荐：humanlike）
5. 点击"保存默认配置"

### 使用示例

#### 场景 1：使用默认配置（简化配置）

```
小旋风 post 格式: txt={word}&sim=1
```

系统自动使用你配置的默认模型和模式

#### 场景 2：覆盖默认配置（灵活调整）

```
小旋风 post 格式: txt={word}&mode=deep&model=claude-opus-4-6&sim=1
```

使用指定的模式和模型，忽略默认配置

#### 场景 3：部分覆盖

只指定模式：
```
txt={word}&mode=creative&sim=1
```
使用 creative 模式 + 默认模型

只指定模型：
```
txt={word}&model=claude-haiku-4-5-20251001&sim=1
```
使用默认模式 + Haiku 模型

### 配置优先级

```
指定的参数 > 用户默认配置 > 系统默认值
```

## 📝 常用配置

### 推荐配置（日常使用）⭐

- **默认模型**：Claude Sonnet 4.5
- **默认模式**：humanlike 或 seo_spider
- **优点**：性能和成本平衡，降低 AI 检测风险，不会超时

### 大量采集（追求速度）

- **默认模型**：Claude Haiku 4.5
- **默认模式**：standard 或 seo_fast
- **优点**：响应速度最快，成本最低，永不超时

### 精品内容（追求质量）

- **默认模型**：Claude Sonnet 4.5（推荐）或 Opus 4.6（仅短文本）
- **默认模式**：deep 或 seo_quality
- **优点**：改写质量最高，内容更丰富
- **注意**：Opus 4.6 仅适合 < 1000 字的短文本，长文本会超时

## 🎨 改写模式说明

### 反 AI 检测模式（推荐）

- **humanlike** - 仿人类写作，最自然
- **mixed** - 混合改写，多种技巧结合
- **deep** - 深度重写，完全重构内容

### 通用模式

- **standard** - 标准改写，保持原意
- **creative** - 创意改写，更生动有趣
- **formal** - 正式改写，商务/学术风格
- **simple** - 简化改写，通俗易懂
- **translate_cn** - 翻译改写

### 专业模式

- **seo** - SEO 优化
- **storytelling** - 故事化叙述
- **xiaohongshu** - 小红书风格
- **wechat** - 微信公众号风格
- **news** - 新闻报道风格

## 🆘 常见问题

### Q: 如何查看当前的默认配置？

A: 在 Web 后台"配置"页面的"当前配置状态"区域可以看到。

### Q: 默认配置会影响已有的小旋风配置吗？

A: 不会。如果小旋风指定了参数，会优先使用指定的参数。

### Q: 可以为不同用户设置不同的默认配置吗？

A: 可以。每个用户都有独立的默认配置。

### Q: 修改默认配置后需要重启服务吗？

A: 不需要，修改后立即生效。

### Q: 如何测试默认配置是否生效？

A: 运行 `test-defaults.bat` 测试脚本，或查看服务器日志。
