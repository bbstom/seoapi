# 快速开始指南

## 🚀 启动服务

### Windows 系统

```bash
# 方式 1: 使用启动脚本
start.bat

# 方式 2: 手动启动
node server.js
```

### Linux 系统

```bash
# 方式 1: 使用启动脚本
./start.sh

# 方式 2: 手动启动
node server.js
```

## 🔐 首次登录

### 1. 访问登录页面

打开浏览器访问：`http://localhost:8000`

系统会自动跳转到登录页面：`http://localhost:8000/login.html`

### 2. 使用默认管理员账号登录

```
用户名: admin
密码: admin123
```

### 3. 登录成功后

- 系统会自动跳转到主界面
- 右上角显示用户信息和退出按钮
- 可以开始使用文本改写功能

## ⚙️ 配置 Claude API Key

### 1. 进入配置页面

点击顶部的"配置"标签页

### 2. 查看你的 API Key

在"你的 API Key"区域可以看到系统为你生成的 API Key：

```
sk_e94611ef0d679c8ee86d30cfbba3b967cdffade90d778b35599fd8c6f231a104
```

这个 API Key 用于小旋风调用 API 时的认证。

### 3. 配置 Claude API Key

在"Claude API Key"输入框中输入你的 Claude API Key：

```
sk-ant-api03-xxxxx...
```

点击"保存 Claude API Key"按钮。

### 4. 修改密码（推荐）

首次登录后建议立即修改密码：

1. 在配置页面下方找到"修改密码"区域
2. 输入旧密码：`admin123`
3. 输入新密码（至少6位）
4. 点击"修改密码"按钮

## 📝 使用文本改写

### 1. 进入改写页面

点击顶部的"文本改写"标签页

### 2. 输入文本

在"输入文本"框中输入要改写的内容

### 3. 选择改写模式

从下拉框中选择改写模式：

- 标准改写 - 保持原意，改变表达
- 仿人类写作 - 降低 AI 检测风险 ⭐
- 混合改写 - 多种技巧结合 ⭐
- 深度重写 - 完全重构内容 ⭐
- 创意改写 - 更生动有趣的表达
- 正式改写 - 商务/学术风格
- 简化改写 - 通俗易懂
- 翻译改写 - 通过翻译达到改写效果

### 4. 选择模型

推荐使用 Claude 3 Sonnet（平衡性能和成本）

### 5. 开始改写

点击"开始改写"按钮，等待几秒钟即可看到结果

### 6. 复制结果

点击"复制结果"按钮将改写后的文本复制到剪贴板

## 🔧 配置小旋风

### 1. 获取 API Key

在 Web 界面的"配置"页面复制你的 API Key

### 2. 在小旋风中配置

```
API名称: Claude AI 伪原创
API地址: http://localhost:8000/api/rewrite
post格式: txt={word}&mode=humanlike&sim=1
Authorization: sk_e94611ef0d679c8ee86d30cfbba3b967cdffade90d778b35599fd8c6f231a104
成功标志: errcode=0
返回错误的字段: errmsg
伪原创后的内容字段: data
```

**重要：** 将 Authorization 字段的值替换为你自己的 API Key

### 3. 测试 API

在小旋风中测试 API 是否正常工作

## 🎨 自定义改写模式

### 1. 进入模式管理

点击顶部的"模式管理"标签页

### 2. 添加新模式

1. 点击"+ 添加新模式"按钮
2. 填写模式信息：
   - 模式 ID：英文标识，如 `xiaohongshu`
   - 模式名称：中文名称，如 `小红书风格`
   - 简短描述：说明用途
   - 改写指令：告诉 AI 如何改写（最重要）
3. 点击"保存模式"

### 3. 编辑现有模式

点击模式卡片上的"编辑"按钮

### 4. 测试模式

点击"测试"按钮会自动切换到改写页面并填充示例文本

## 👥 多用户管理（管理员）

### 1. 创建新用户

管理员可以为团队成员创建账号：

1. 登录管理员账号
2. 进入"用户管理"标签页（仅管理员可见）
3. 点击"添加用户"
4. 填写用户名、密码、Claude API Key（可选）
5. 系统自动生成该用户的 API Key

### 2. 查看用户列表

可以看到所有用户及其 API Key

### 3. 删除用户

可以删除不需要的用户（不能删除 admin）

## 🔒 安全建议

首次登录后立即修改 admin 密码

如果 API Key 泄露，可以在配置页面重新生成

## 🐛 常见问题

### Q1: 访问 localhost:8000 没有登录窗口？

**A:** 检查以下几点：

1. 确认服务已启动：`Get-Process -Name node`
2. 清除浏览器缓存
3. 尝试访问：`http://localhost:8000/login.html`
4. 检查浏览器控制台是否有错误

### Q2: 登录后提示"会话已过期"？

**A:** 清除浏览器的 localStorage：

```javascript
// 在浏览器控制台执行
localStorage.clear();
location.reload();
```

### Q3: 改写时提示"用户未配置 Claude API Key"？

**A:** 在配置页面输入你的 Claude API Key

### Q4: 小旋风调用 API 失败？

**A:** 检查以下几点：

1. 确认 Authorization 字段填写了正确的 API Key
2. 确认 API 地址正确
3. 确认 post 格式为：`txt={word}&mode=humanlike&sim=1`
4. 确认成功标志为：`errcode=0`
5. 确认返回字段为：`data`

### Q5: 忘记密码怎么办？

**A:** 删除 `data/users.json` 文件，重启服务会重新创建默认管理员账号
