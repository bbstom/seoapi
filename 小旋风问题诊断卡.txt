╔══════════════════════════════════════════════════════════════╗
║          小旋风改写失败 - 快速诊断卡                        ║
╚══════════════════════════════════════════════════════════════╝

┌──────────────────────────────────────────────────────────────┐
│ 问题症状                                                     │
├──────────────────────────────────────────────────────────────┤
│ ✅ 短文本（<100 字符）改写成功                              │
│ ❌ 长文本（>1000 字符）改写失败                             │
│ ✅ 服务器日志显示成功（errcode=0）                          │
│ ❌ 小旋风显示"改写失败"                                     │
└──────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│ 根本原因                                                     │
├──────────────────────────────────────────────────────────────┤
│ 小旋风对 API 响应体有大小限制（约 1300-1500 字符）         │
│ 超过限制时，小旋风无法正确解析响应                          │
└──────────────────────────────────────────────────────────────┘

╔══════════════════════════════════════════════════════════════╗
║ 解决方案（3 选 1）                                          ║
╚══════════════════════════════════════════════════════════════╝

┌──────────────────────────────────────────────────────────────┐
│ 方案 1：限制采集长度（推荐）⭐                              │
├──────────────────────────────────────────────────────────────┤
│ 在小旋风采集规则中：                                        │
│   内容截取：前 800 字符                                     │
│                                                              │
│ 优点：最简单有效，100% 解决问题                             │
└──────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│ 方案 2：添加 compact 参数                                   │
├──────────────────────────────────────────────────────────────┤
│ 在小旋风 post 格式中：                                      │
│   txt={word}&mode=seo_fast&sim=1&compact=1                  │
│                                                              │
│ 效果：压缩输出，减少 10-20% 大小                            │
└──────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│ 方案 3：分段改写                                            │
├──────────────────────────────────────────────────────────────┤
│ 在采集规则中分段：                                          │
│   段落1：前 800 字符                                        │
│   段落2：801-1600 字符                                      │
│                                                              │
│ 适用：超长文章（>2000 字符）                                │
└──────────────────────────────────────────────────────────────┘

╔══════════════════════════════════════════════════════════════╗
║ 文本长度安全范围                                            ║
╚══════════════════════════════════════════════════════════════╝

  <800 字符     ████████████████████ 100% ✅ 安全
  800-1000      ████████████████░░░░  95% ✅ 安全
  1000-1200     ██████████░░░░░░░░░░  70% ⚠️ 可能失败
  >1200         ████░░░░░░░░░░░░░░░░  30% ❌ 很可能失败

╔══════════════════════════════════════════════════════════════╗
║ 推荐配置                                                    ║
╚══════════════════════════════════════════════════════════════╝

API名称: Claude AI 伪原创
API地址: http://your-domain.com/api/rewrite
post格式: txt={word}&mode=seo_fast&sim=1&compact=1
Authorization: your_api_key
成功标志: errcode=0
返回错误的字段: errmsg
伪原创后的内容字段: data

采集规则：
  内容截取：前 800 字符 ⭐

╔══════════════════════════════════════════════════════════════╗
║ 快速诊断步骤                                                ║
╚══════════════════════════════════════════════════════════════╝

1️⃣ 查看服务器日志
   [请求 xxx] 响应内容长度: ??? 字符

2️⃣ 判断原因
   长度 < 1200 → 不是长度问题，检查其他原因
   长度 > 1200 → 是长度问题 ⚠️

3️⃣ 应用解决方案
   → 限制采集长度为 800 字符（最简单）
   → 或添加 compact=1 参数（次选）
