========================================
小旋风 API 配置 - 复制粘贴即可使用
========================================

API名称: Claude AI 伪原创

API地址: http://localhost:8000/api/rewrite

post格式: txt={word}&mode=seo_fast&sim=1

Authorization: sk_f7747efeefece9fbeb79bfecd825e01ff427c2f343acd398f6f3713469906d00

成功标志: errcode=0

返回错误的字段: errmsg

伪原创后的内容字段: data

========================================
重要提示
========================================

1. 确保服务正在运行
   命令: cd seoapi-nodejs && node server.js

2. 如果是远程服务器，将 localhost 改为服务器 IP
   例如: http://192.168.1.100:8000/api/rewrite

3. Authorization 是你的 API Key
   查看位置: seoapi-nodejs/data/users.json

4. 支持的改写模式：
   - seo_fast (快速SEO改写) ⭐ 推荐
   - seo_spider (SEO蜘蛛优化)
   - seo_deep (深度伪原创)
   - seo_quality (高质量改写)
   - seo_original (原创风格)
   - humanlike (仿人类写作)
   - mixed (混合改写)
   - deep (深度重写)

5. 可选：指定模型
   post格式: txt={word}&mode=seo_fast&model=claude-sonnet-4-5-20250929&sim=1

========================================
测试配置
========================================

1. 启动服务后，访问: http://localhost:8000/health
   应该返回: {"status":"ok"}

2. 在小旋风中点击"测试"按钮
   应该返回改写后的文本

3. 如果返回"空内容"，查看:
   - 小旋风故障排查指南.md
   - 快速诊断卡片.md
