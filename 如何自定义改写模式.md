# 如何自定义改写模式

## 📝 改写模式是什么？

改写模式就是告诉 Claude AI **如何改写文本的指令**。

比如：
- "标准改写" = 告诉 AI "保持原意，改变表达"
- "仿人类写作" = 告诉 AI "用口语化、不完美的方式改写"

## 🎨 如何修改现有模式？

### 步骤 1: 打开配置文件

```bash
编辑文件: config/rewrite-modes.js
```

### 步骤 2: 找到要修改的模式

```javascript
humanlike: {
    name: '仿人类写作',
    description: '降低 AI 检测风险',
    antiAI: true,
    prompt: `请对以下文本进行改写，要求改写后的内容更像人类写作风格：
1. 使用口语化表达，避免过于正式
2. 适当添加一些个人观点或感受
...`
}
```

### 步骤 3: 修改 prompt（提示词）

```javascript
humanlike: {
    name: '仿人类写作',
    description: '降低 AI 检测风险',
    antiAI: true,
    prompt: `请对以下文本进行改写，要求：
1. 使用大白话，像聊天一样
2. 多用"其实"、"不过"、"比如"这些词
3. 句子有长有短，不要太工整
4. 可以加点个人看法
5. 不要太正式，轻松点`  // ← 你的自定义指令
}
```

### 步骤 4: 重启服务

```bash
# Windows
按 Ctrl+C 停止
start.bat

# Linux
./stop.sh
./start.sh
```

## 🔧 配置文件结构说明

```javascript
模式ID: {
    name: '显示名称',           // 在界面上显示的名字
    description: '简短描述',    // 一句话说明这个模式的用途
    antiAI: true,              // 可选，标记是否为反AI检测模式
    prompt: `给 AI 的指令`      // 核心：告诉 AI 如何改写
}
```

### 参数说明

| 参数 | 必填 | 说明 | 示例 |
|------|------|------|------|
| name | ✅ | 模式名称 | "标准改写" |
| description | ✅ | 简短描述 | "保持原意，改变表达" |
| antiAI | ❌ | 是否反AI检测 | true / false |
| prompt | ✅ | AI 指令 | "请对以下文本..." |

## 💡 编写 Prompt 的技巧

### 1. 要具体明确

❌ 不好的例子：
```javascript
prompt: '请改写这段文字'
```

✅ 好的例子：
```javascript
prompt: `请改写这段文字，要求：
1. 改变句式结构
2. 使用同义词
3. 保持专业性`
```

### 2. 使用编号列表

```javascript
prompt: `请改写，要求：
1. 第一点要求
2. 第二点要求
3. 第三点要求`
```

### 3. 给出具体例子

```javascript
prompt: `请改写成口语化风格：
1. 使用"其实"、"不过"、"比如"等词
2. 像这样："其实这个方法挺简单的"
3. 而不是："该方法较为简单"`
```

### 4. 说明不要做什么

```javascript
prompt: `请改写，要求：
1. 不要使用过于正式的词汇
2. 不要使用专业术语
3. 避免长句子（超过30字）`
```

### 5. 指定风格参考

```javascript
prompt: `请改写成类似知乎高赞回答的风格：
1. 开头先抛出观点
2. 然后举例说明
3. 最后总结升华
4. 语气亲切但专业`
```

## 🧪 测试你的自定义模式

### 方法 1: Web 界面测试

1. 启动服务
2. 打开 http://localhost:8000
3. 在"改写模式"下拉框中选择你的新模式
4. 输入测试文本
5. 点击"开始改写"

### 方法 2: API 测试

```bash
curl -X POST http://localhost:8000/api/rewrite \
  -H "Content-Type: application/json" \
  -d '{
    "txt": "测试文本",
    "mode": "xiaohongshu"
  }'
```

## ⚙️ 高级技巧

### 1. 组合多个要求

```javascript
prompt: `请改写，同时满足以下要求：
【风格】口语化、轻松
【长度】比原文增加20%
【结构】分3-5段
【特点】添加emoji，使用短句`
```

### 2. 指定目标受众

```javascript
prompt: `请改写，目标读者是：
- 年龄：25-35岁
- 职业：互联网从业者
- 需求：快速获取信息
要求：专业但不枯燥，有趣但不低俗`
```

### 3. 设定改写程度

```javascript
prompt: `请进行【深度改写】：
- 相似度控制在30%以下
- 完全改变句式结构
- 用不同的例子和比喻
- 保持核心观点不变`
```

## 🔄 动态加载配置（高级）

如果你想在不重启服务的情况下更新模式，可以添加热重载功能：

```javascript
// 在 server.js 中添加
app.post('/api/reload-modes', (req, res) => {
    delete require.cache[require.resolve('./config/rewrite-modes')];
    const newModes = require('./config/rewrite-modes');
    
    // 重新构建 REWRITE_PROMPTS
    for (const [key, value] of Object.entries(newModes)) {
        REWRITE_PROMPTS[key] = value.prompt;
    }
    
    res.json({ success: true, message: '模式已重新加载' });
});
```

使用：
```bash
curl -X POST http://localhost:8000/api/reload-modes
```
