# iflow.cn 配置使用指南

## 快速配置步骤

### 第一步：添加 iflow API 配置

1. 登录系统
2. 点击顶部导航栏的"系统配置"
3. 点击"外部API配置"子标签
4. 点击"添加配置"按钮
5. 填写以下信息：
   - **配置名称**：`iflow`（或任意名称）
   - **API 地址**：`https://apis.iflow.cn/v1`
   - **API Key**：你的 iflow API Key
   - **API 类型**：选择 `openai`
6. 点击"测试连接"按钮
7. 等待测试完成，应该显示"连接成功，获取到 X 个模型"
8. 点击"保存"按钮

### 第二步：设置默认调用配置

1. 在"系统配置"页面，点击"默认调用配置"子标签
2. 在"选择 API 配置"下拉列表中选择 `iflow`
3. 等待模型列表加载完成
4. 在"默认模型"下拉列表中选择你想使用的模型（例如 `qwen3-235b`）
5. 点击"保存配置"按钮

### 第三步：测试改写功能

1. 点击顶部导航栏的"文本改写"
2. 在文本框中输入测试文本，例如：
   ```
   这是一段测试文本，用于验证 iflow API 是否配置成功。
   ```
3. 选择改写模式（例如"人性化改写"）
4. 点击"开始改写"按钮
5. 等待改写完成

## 验证配置是否生效

### 方法1：查看后台日志

如果你有服务器访问权限，可以查看后台日志：

## 常见问题

### Q1: 测试连接成功，但改写时报错 "invalid x-api-key"

**原因**：可能没有设置默认调用配置。

**解决方法**：
1. 进入"系统配置-默认调用配置"
2. 选择你的 iflow 配置
3. 选择默认模型
4. 保存配置

### Q2: 找不到模型列表

**原因**：可能是 API 地址不正确或 API Key 无效。

**解决方法**：
1. 确认 API 地址是 `https://apis.iflow.cn/v1`（注意末尾的 `/v1`）
2. 确认 API Key 是否正确
3. 重新测试连接

### Q3: 改写速度很慢

**原因**：iflow 的响应速度取决于网络和服务器负载。

**解决方法**：
- 耐心等待
- 如果长时间无响应，检查网络连接
- 尝试选择不同的模型

### Q4: 想切换回其他 API

**解决方法**：
1. 进入"系统配置-外部API配置"
2. 添加新的 API 配置（例如 Claude 官方）
3. 进入"默认调用配置"
4. 选择新的 API 配置
5. 保存配置

## iflow 支持的模型示例

根据测试连接时获取的模型列表，iflow 可能支持以下模型（具体以实际获取为准）：

- `qwen3-235b` - 通义千问 3 235B
- `qwen-turbo` - 通义千问 Turbo
- `qwen-plus` - 通义千问 Plus
- `qwen-max` - 通义千问 Max
- 其他模型...

选择模型时，建议：
- **速度优先**：选择 Turbo 版本
- **质量优先**：选择 Max 或大参数版本
- **平衡选择**：选择 Plus 版本

## API 地址说明

iflow.cn 的 API 地址格式：

```
https://apis.iflow.cn/v1
```

注意事项：
- ✅ 必须包含 `/v1` 后缀
- ✅ 使用 HTTPS 协议
- ❌ 不要添加额外的路径（如 `/chat/completions`）
- ❌ 不要在末尾添加斜杠

## 兼容性说明

iflow.cn 使用 OpenAI 兼容格式，系统会自动识别并适配。你不需要做任何特殊配置。

支持的功能：
- ✅ 文本改写
- ✅ 多种改写模式
- ✅ 长文本分段处理
- ✅ 相似度计算
- ✅ 日志记录
