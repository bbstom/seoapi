# API节点弹窗优化 - 说明文档

## 优化内容

将新增/编辑API节点的表单从页面内嵌改为弹窗模式，提升用户体验。

## 修改内容

### 前端修改
**文件：** `frontend/src/pages/SystemPage.jsx`

1. **弹窗容器**
   - 添加全屏遮罩层（半透明黑色背景）
   - 居中显示弹窗
   - 最大宽度 4xl，最大高度 90vh
   - 支持滚动查看完整表单

2. **交互优化**
   - 点击背景关闭弹窗
   - 点击右上角 X 按钮关闭
   - 点击底部"取消"按钮关闭
   - 添加弹窗动画效果

3. **样式调整**
   - 移除表单原有的卡片边框
   - 增大标题字号（xl）
   - 增大关闭按钮尺寸
   - 调整按钮布局（取消在左，保存在右）
   - 增大按钮高度和圆角

## 优势对比

### 修改前（页面内嵌）
- ❌ 表单展开后占用大量页面空间
- ❌ 需要滚动页面查看其他节点
- ❌ 视觉焦点不集中
- ❌ 取消后需要重新定位

### 修改后（弹窗模式）
- ✅ 不影响页面布局
- ✅ 焦点集中在表单上
- ✅ 背景虚化，突出表单
- ✅ 关闭后自动回到原位置
- ✅ 更符合现代UI设计规范

## 功能特性

### 1. 多种关闭方式
- 点击背景遮罩
- 点击右上角 X 按钮
- 点击底部"取消"按钮
- 按 ESC 键（浏览器默认行为）

### 2. 响应式设计
- 桌面端：最大宽度 4xl（56rem）
- 移动端：自适应屏幕宽度，保留边距
- 高度自适应，超出时可滚动

### 3. 动画效果
- 弹窗淡入效果（animate-in）
- 背景渐变显示
- 流畅的过渡动画

## 验证方法

1. 访问 https://api.vpno.eu.org/
2. 进入"API 管理"页面
3. 点击"添加节点"按钮
4. 应该看到居中的弹窗，背景变暗
5. 尝试以下操作：
   - 点击背景 → 弹窗关闭
   - 点击右上角 X → 弹窗关闭
   - 点击底部"取消" → 弹窗关闭
   - 填写表单后点击"添加节点" → 保存并关闭
6. 点击某个节点的"编辑"按钮
7. 应该看到相同的弹窗，标题显示"编辑 AI 节点"

## 技术细节

### 弹窗结构
```jsx
<div className="fixed inset-0 bg-black/50 ...">  {/* 遮罩层 */}
  <div className="bg-white rounded-2xl ...">      {/* 弹窗容器 */}
    <ApiConfigForm ... />                         {/* 表单组件 */}
  </div>
</div>
```

### 关闭逻辑
```jsx
onClick={(e) => {
  if (e.target === e.currentTarget) {  // 只在点击背景时触发
    setIsAdding(false);
    setEditingConfig(null);
  }
}}
```

### CSS 类说明
- `fixed inset-0` - 全屏固定定位
- `bg-black/50` - 半透明黑色背景
- `z-50` - 高层级，覆盖其他内容
- `max-h-[90vh]` - 最大高度 90% 视口
- `overflow-y-auto` - 内容超出时可滚动
- `animate-in` - Tailwind 动画类
