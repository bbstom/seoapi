## 💡 常见问题

### Q1: 提示"无效的 apikey"？

**A:** 确认使用的是正确的 API Key：
```
sk_f7747efeefece9fbeb79bfecd825e01ff427c2f343acd398f6f3713469906d00
```

### Q2: 返回空内容？

**A:** 检查：
1. 成功标志是否设置为 `errcode=0`（字符串）
2. 内容字段是否设置为 `data`
3. 是否配置了 Claude API Key

### Q3: 如何查看我的 API Key？

**A:** 
1. 登录系统 `http://localhost:8000`
2. 进入"配置"标签页
3. 查看"你的 API Key"区域

### Q4: 可以重新生成 API Key 吗？

**A:** 可以，在配置页面点击"重新生成"按钮。

### Q5: 支持多个用户吗？

**A:** 支持。管理员可以创建多个用户，每个用户有独立的 API Key。
