# 小旋风配置教程

## 🎯 基础配置

### 配置参数

```
API名称: Claude AI 伪原创
API地址: http://your-server:8000/api/rewrite
post格式: txt={word}&mode=humanlike&sim=1
Authorization: your_api_key_here
成功标志: errcode=0
返回错误的字段: errmsg
伪原创后的内容字段: data
```

### 参数说明

| 参数 | 说明 | 必填 | 示例 |
|------|------|------|------|
| txt | 要改写的文本（小旋风变量：{word}） | 是 | txt={word} |
| mode | 改写模式 | 否 | mode=humanlike |
| model | Claude 模型 | 否 | model=claude-sonnet-4-5-20250929 |
| sim | 是否返回相似度（1=返回，0=不返回） | 否 | sim=1 |

## 🎨 自定义模型

### 方式 1: 在 post 格式中指定模型（推荐）

```
post格式: txt={word}&mode=humanlike&model=claude-sonnet-4-5-20250929&sim=1
```

### 方式 2: 使用默认模型

如果不指定 model 参数，系统会使用默认模型：`claude-sonnet-4-5-20250929`

### 支持的模型列表

| 模型 ID | 名称 | 特点 | 推荐场景 |
|---------|------|------|---------|
| claude-haiku-4-5-20251001 | Claude Haiku 4.5 | 最快速 | 大量采集，追求速度 |
| claude-sonnet-4-5-20250929 | Claude Sonnet 4.5 | 平衡（推荐）⭐ | 日常使用 |
| claude-sonnet-4-5-20250929-thinking | Claude Sonnet 4.5 Thinking | 深入分析 | 复杂内容 |
| claude-sonnet-4-6 | Claude Sonnet 4.6 | 最新版 | 追求新功能 |
| claude-opus-4-5 | Claude Opus 4.5 | 强大 | 高质量要求 |
| claude-opus-4-5-20251101-thinking | Claude Opus 4.5 Thinking | 最强分析 | 最复杂任务 |
| claude-opus-4-6 | Claude Opus 4.6 | 顶级 | 极致质量 |

## 🔧 改写模式说明

| 模式 ID | 名称 | 说明 | 推荐搭配模型 |
|---------|------|------|-------------|
| standard | 标准改写 | 保持原意，改变表达 | Sonnet 4.5 |
| humanlike | 仿人类写作 | 降低 AI 检测风险 ⭐ | Sonnet 4.5 |
| mixed | 混合改写 | 多种技巧结合 ⭐ | Sonnet 4.5 |
| deep | 深度重写 | 完全重构内容 ⭐ | Opus 4.5 |
| creative | 创意改写 | 更生动有趣 | Sonnet 4.5 |
| formal | 正式改写 | 商务/学术风格 | Opus 4.5 |
| simple | 简化改写 | 通俗易懂 | Haiku 4.5 |
| translate_cn | 翻译改写 | 通过翻译改写 | Sonnet 4.5 |

⭐ 标记的模式可以降低 AI 检测风险

## 💡 最佳实践

### 1. 根据内容类型选择模型

**新闻资讯类**
```
model=claude-haiku-4-5-20251001&mode=standard
```
- 速度快，适合大量采集

**营销文案类**
```
model=claude-sonnet-4-5-20250929&mode=creative
```
- 平衡质量和速度

**专业文章类**
```
model=claude-opus-4-6&mode=formal
```
- 高质量，适合重要内容

**规避 AI 检测**
```
model=claude-sonnet-4-5-20250929&mode=humanlike
```
- 降低 AI 检测风险

### 2. 根据采集量选择模型

**大量采集（>1000篇/天）**
- 推荐：Haiku 4.5
- 原因：速度快，成本低

**中等采集（100-1000篇/天）**
- 推荐：Sonnet 4.5
- 原因：质量和速度平衡

**精品采集（<100篇/天）**
- 推荐：Opus 4.6
- 原因：质量最高

### 3. 根据预算选择模型

所有模型价格相同（$10/$50 per M tokens），但不同模型的 token 消耗可能不同：

**最省 tokens**: Haiku 4.5（响应简洁）
**平衡**: Sonnet 4.5（推荐）
**最多 tokens**: Opus 4.6 + Thinking（分析详细）

## ⚠️ 常见问题

### Q1: 如何在小旋风中切换模型？

**A:** 修改 post 格式中的 model 参数：
```
txt={word}&mode=humanlike&model=claude-opus-4-6&sim=1
```

### Q2: 不同模型的改写质量差异大吗？

**A:** 
- Haiku：快速，质量良好，适合简单改写
- Sonnet：平衡，质量优秀，适合大多数场景
- Opus：最强，质量最高，适合重要内容

### Q3: Thinking 模式有什么区别？

**A:** Thinking 模式会进行更深入的思考和分析：
- 优点：改写质量更高，更符合人类写作习惯
- 缺点：响应时间稍长（多1-2秒）
- 适合：复杂内容、重要文章

### Q4: 如何选择最合适的模型？

**A:** 根据需求选择：
1. **追求速度** → Haiku 4.5
2. **日常使用** → Sonnet 4.5（推荐）
3. **追求质量** → Opus 4.6
4. **规避检测** → Sonnet 4.5 + humanlike 模式

### Q5: 可以为不同的采集任务配置不同的模型吗？

**A:** 可以！在小旋风中创建多个 API 配置：
- API 1: 新闻采集 → Haiku 4.5 + standard
- API 2: 文章采集 → Sonnet 4.5 + humanlike
- API 3: 精品采集 → Opus 4.6 + deep
