# SEO 养站优化指南

## 🎯 目标说明

本系统专为 SEO 养站优化，主要目标：
1. **避免查重** - 深度改写，降低相似度
2. **避免搜索引擎惩罚** - 保持内容质量
3. **吸引蜘蛛** - 优化内容结构和关键词
4. **提高收录** - 增加蜘蛛访问频率

## 🚀 新增 SEO 专用模式

### 1. SEO 蜘蛛优化（seo_spider）⭐ 推荐

**适用场景**：日常养站，平衡质量和效率

**特点**：
- 相似度控制在 20% 以下
- 优化关键词分布和密度（2-3%）
- 段落结构清晰，便于蜘蛛抓取
- 避免 AI 痕迹，使用人类化表达

**使用方法**：
```
小旋风配置: txt={word}&mode=seo_spider&sim=1
```

**效果**：
- ✅ 有效避免查重
- ✅ 吸引蜘蛛爬取
- ✅ 提高收录率
- ✅ 避免被搜索引擎惩罚

### 2. SEO 深度伪原创（seo_deep）⭐⭐ 高质量

**适用场景**：重要内容，追求极低相似度

**特点**：
- 相似度 < 15%，极低查重风险
- 完全重构文章结构和逻辑
- 增加新观点、例子和数据
- 高质量原创化改写

**使用方法**：
```
小旋风配置: txt={word}&mode=seo_deep&sim=1
```

**效果**：
- ✅ 几乎无法被查重
- ✅ 内容质量高
- ✅ 搜索引擎友好
- ✅ 提升页面权重

### 3. SEO 高质量改写（seo_quality）⭐⭐⭐ 最佳

**适用场景**：精品内容，追求最高收录率

**特点**：
- 扩充内容 30-50%，增加信息量
- 添加小标题、数据、案例
- 提升 E-A-T（专业性、权威性、可信度）
- 优化用户体验和停留时间

**使用方法**：
```
小旋风配置: txt={word}&mode=seo_quality&sim=1
```

**效果**：
- ✅ 内容质量最高
- ✅ 收录率最高
- ✅ 排名提升明显
- ✅ 用户体验好

### 4. SEO 快速改写（seo_fast）

**适用场景**：大量采集，批量处理

**特点**：
- 快速改写，适合批量处理
- 相似度 30-40%
- 保持基本质量
- 效率优先

**使用方法**：
```
小旋风配置: txt={word}&mode=seo_fast&sim=1
```

**效果**：
- ✅ 处理速度快
- ✅ 成本低
- ✅ 适合大量内容
- ⚠️ 质量中等

### 5. SEO 原创风格（seo_original）⭐⭐⭐ 顶级

**适用场景**：核心页面，追求完全原创

**特点**：
- 相似度 < 10%，接近完全原创
- 模拟人工撰写风格
- 添加独特观点和见解
- 最高质量改写

**使用方法**：
```
小旋风配置: txt={word}&mode=seo_original&sim=1
```

**效果**：
- ✅ 完全原创化
- ✅ 搜索引擎最友好
- ✅ 长期排名稳定
- ✅ 品牌价值高

**原因**：
- 处理速度快

## ⚠️ 超时问题注意

### 模型速度与文本长度

| 模型 | 速度 | 适用文本长度 | 超时风险 |
|------|------|--------------|----------|
| Haiku 4.5 | ⚡⚡⚡⚡⚡ | 0-10000 字 | 🟢 无风险 |
| Sonnet 4.5 | ⚡⚡⚡⚡ | 0-5000 字 | 🟢 低风险 |
| Sonnet 4.6 | ⚡⚡⚡ | 0-3000 字 | 🟡 中风险 |
| Opus 4.5 | ⚡⚡ | 0-2000 字 | 🟡 中风险 |
| Opus 4.6 | ⚡ | 0-1000 字 | 🔴 高风险 |

### 重要提示

- ⚠️ **Opus 4.6 不适合长文本**（> 2000 字会超时）
- ✅ **Sonnet 4.5 是最佳选择**（速度快，质量好）
- ✅ **Haiku 4.5 适合批量处理**（永不超时）
- 📖 详细说明请查看 [超时问题解决方案.md](./超时问题解决方案.md)
