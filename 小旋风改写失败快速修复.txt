========================================
📋 响应格式变化
========================================

旧格式（可能导致失败）：
{
  "errcode": "0",
  "data": "...",
  "_meta": {...}  ← 小旋风可能不支持
}

新格式（兼容性更好）：
{
  "errcode": "0",
  "data": "..."
}

========================================
💡 如果需要查看元数据
========================================

添加 debug=1 参数：
curl https://api.vpno.eu.org/api/rewrite?debug=1 \
  -d "txt=测试&mode=seo_fast"

返回完整信息：
{
  "errcode": "0",
  "data": "...",
  "_meta": {
    "mode": "seo_fast",
    "model": "claude-opus-4-6",
    "duration": 5.2,
    "usage": {...}
  }
}

========================================
🎯 推荐配置
========================================

速度优先（推荐）：
post格式: txt={word}&mode=seo_fast&model=claude-haiku-4-5-20251001&sim=1

平衡配置：
post格式: txt={word}&mode=seo_fast&model=claude-sonnet-4-5-20250929&sim=1

质量优先：
post格式: txt={word}&mode=seo_deep&model=claude-opus-4-6&sim=1

========================================
⚠️ 注意事项
========================================

1. Opus 模型处理长文本（>1000字）较慢
   推荐使用 Sonnet 或 Haiku

2. 文本长度建议：
   < 500 字 → 任何模型
   500-2000 字 → Sonnet
   > 2000 字 → Haiku

3. 在 Web 后台设置默认模型
   避免每次都指定模型参数

========================================
🔍 故障排查
========================================

如果仍然失败：

1. 查看服务器日志
pm2 logs seoapi --lines 50

2. 检查是否显示"改写完成"
   如果显示，说明 API 正常

3. 使用 curl 直接测试
   查看返回的 JSON 格式

4. 检查小旋风配置
   - API 地址正确
   - Authorization 正确
   - 成功标志: errcode=0
   - 返回字段: data
